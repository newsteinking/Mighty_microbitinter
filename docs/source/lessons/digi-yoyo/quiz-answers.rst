3. Create a ``while loop`` that will loop until the **variable** ``count`` equals 4.
------------------------------------------------------------------------------------------------


.. image:: /static/mb/lessons/digi-yoyo-0.png
   :target: /static/mb/lessons/digi-yoyo-0.png
   :alt: 


:raw-html-m2r:`<br/>`

.. code-block:: blocks

   let count = 0
   while (count < 5) {
       count = count + 1
   }
